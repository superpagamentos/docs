---
title: "Guia de Erros"
---

Este documento lista todos os possíveis códigos de erro que podem ser retornados pelas APIs da Super. Os erros são retornados no parâmetro `returnCode` e estão organizados por categorias para facilitar a identificação e resolução de problemas.

## Erros de Autenticação

| Código | Descrição | Causa Provável |
|--------|-----------|----------------|
| -1000 | USER_NOT_FOUND | Usuário não encontrado no sistema |
| -1100 | USER_NOT_PERMITTED | Usuário não está permitido |
| -1200 | INVALID_KEYS | Chaves de autenticação inválidas |
| -1300 | INVALID_REFRESHTOKEN | Token de atualização inválido |
| -1400 | INVALID_PHONE_NUMBER | Número de telefone inválido |
| -1500 | INVALID_RESET_TOKEN_PASSWORD | Token de redefinição de senha inválido |
| -1800 | COMPANY_NOT_APPROVED | Empresa ainda não foi aprovada |

## Erros do Sistema

|--------|-----------|----------------|
| -9999 | UNEXPECTED_ERROR | Erro inesperado no sistema |
| -7000 | UNEXPECTED_PARAMS_ERROR | Erro inesperado em parâmetros do sistema |
